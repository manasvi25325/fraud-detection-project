# 💳 Fraud Detection System

## 📌 Project Overview
This project detects fraudulent financial transactions using Machine Learning.

## 🚀 Features
- Detects fraud transactions
- Calculates risk score
- Real-time prediction using API
- User-friendly UI using Streamlit

## 🛠️ Technologies Used
- Python
- Flask (API)
- Streamlit (UI)
- Scikit-learn
- Pandas, NumPy

## ▶️ How to Run

### Step 1: Install dependencies
pip install -r requirements.txt

### Step 2: Run API
python app.py

### Step 3: Run UI
streamlit run dashboard.py

## 📊 Model
- Machine Learning model trained for fraud detection
- Uses transaction features and behavior analysis
